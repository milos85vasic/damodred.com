== Description ==

Juliet is an elegant, blog-focused, feminine theme perfect for lifestyle blogging. The front page allows you to showcase featured posts. Juliet is the perfect blogging theme with a wide range of features for customizing the layout.

* Off Canvas Menus
* Fancy Scrollbar
* Front Page Banner
* Front Page Featured Posts		
* Image and Text Logos		
* Background Image		
* Background Color	
* Special Sticky Posts Format with Signature	
* Optional Full Width Posts and Pages		
* Excerpt or Full Blog Feed Layouts
* Integration with Recent Posts Widget With Thumbnails
* Integration with MailPoet
* Easy to Create Social Media Icon Menus
* Header and Footer Area Widgets
* Extra Page, Post and Front Page Specific Sidebars		
* Footer Widgets
* Full Width Footer Widget Area
